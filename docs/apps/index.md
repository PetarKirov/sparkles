# Apps

Repository apps and scripts are documented here.

- [run-examples](./run-examples/) - Runs the `core-cli` single-file examples.
- [run-tests](./run-tests/) - Runs unit tests across all Sparkles subpackages.
- [run-md-examples](./run-md-examples/) - Extracts runnable D examples from Markdown and verifies or updates expected output blocks.

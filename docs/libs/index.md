# Libraries

Sparkles is organized as D `dub` subpackages under `libs/`.

- [core-cli](./core-cli/) - Terminal styling, pretty-printing, CLI helpers, and `@nogc`-friendly text utilities.
- [markdown](./markdown/) - Markdown parsing, rendering, and conformance-testing helpers.
- [math](./math/) - Math primitives for games and graphics.
- [test-utils](./test-utils/) - Test helpers for diffs, temporary filesystems, and strings.

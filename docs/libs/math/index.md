# math

`sparkles:math` provides math primitives for games and graphics.

## Scope

- Fixed-size vector types
- Optional named fields
- Swizzling support
- Component-wise arithmetic and scalar operations

## Package

Add the library as a `dub` dependency:

```sdl
dependency "sparkles:math" version="*"
```

## Source Layout

- `libs/math/src/sparkles/math/package.d` - package exports
- `libs/math/src/sparkles/math/vector.d` - vector implementation and aliases

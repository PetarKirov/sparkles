- **#D1 (minor markup normalization) — `prisma.md` Overview quote of `README.md:34-36`.** The
  README bullets are markdown links (`- [**Prisma Client**](https://…): Auto-generated and
type-safe query builder…`). The page quotes them with the link URL stripped
  (`- **Prisma Client**: Auto-generated…`). The visible prose is verbatim; only the link
  target is dropped. Consistent with how the corpus normalizes markdown link markup inside
  quotes; not a factual change.
- **No substantive discrepancies.** Every mechanism claim resolves to source. The page's
  central architectural framing — the Rust engine narrowed to a WASM **query compiler** with
  execution moved to a TS interpreter over a mandatory driver adapter — is grounded in
  `ClientEngine.ts` + `query-interpreter.ts` + `driver-adapter-utils/types.ts` as the **only**
  engine in the tree; the older native library/binary engines are asserted only as
  historical/comment-attested (rows #29, #56), correctly flagged.

## Discrepancies

- **#D1 ⚠ (README vs code) — `README.md:1087` vs `src/index.js:515-517`.** The README says the
  connection max-lifetime is _"a random interval for each connection between 45 and 90 minutes."_
  The code computes `max_lifetime = () => 60 * (30 + Math.random() * 30)` **seconds**, i.e.
  `1800`–`3600` s = **30–60 minutes**, not 45–90. The deep-dive avoids the wrong figure: it
  quotes the code expression and says "random `max_lifetime`", not the README's minutes. Flagged
  here as a stale README fact; the mechanism (randomized per-connection recycling to bound
  server-side prepared-statement memory) is correct either way.

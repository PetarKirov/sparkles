  quotes. Both fragments are verbatim; the polarity of the original ("should NOT be
  returned if…") is preserved. (An earlier draft inserted a `[no]` bracket inside the quote;
  it was rewritten to avoid altering the quoted text.)

  including accessing embedded struct attributes" and elides the trailing "and the ability
  to use functions and struct tags to customize field names" (`reflect.go:4-5`). Standard
  ellipsis condensation; no meaning change.

  README's H2 header calls Exposed "an ORM framework for Kotlin" (claim 2) while the very
  next paragraph calls the core "a lightweight SQL library" and the DAO "lightweight" (claim
  1). The page surfaces this tension deliberately (Overview §Design philosophy) rather than
  picking one; both strings are verbatim from `README.md`. No contradiction introduced.

  Exposed, an ORM framework for Kotlin") with the source's `**bold**` and

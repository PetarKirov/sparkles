## Discrepancies

- **None substantive.** All verbatim prose/docstring/CHANGELOG quotes and every code/type
  claim match the pinned tree. Disclosed normalizations are typographic only: two-line haddock
  docstrings joined into one sentence (#8, #33), leading-"The" lowercased (#8), and the Short

- **API-history note (not a page error).** The page describes the _current_ (`1.10`) API —
  `Connection.use` as the runner and `preparable`/`unpreparable` constructors. Older hasql
  releases exposed `Hasql.Session.run session connection` and a `Statement sql enc dec prepared`
  constructor with a boolean flag; the pinned tree no longer does (`Hasql.Session` exports no
  `run`; the internal `Session.run` is used by `Connection.use`). The page reflects the pinned
  source, not the historical shape — flagged here because prior briefs/tutorials describe the
  old API.

## Discrepancies

**No factual ⚠ discrepancies.** The catalog table, the cubic-wall callout, both
cross-cutting splits, and the source spine are all internally consistent with the seven
theory deep-dives and their grounding ledgers. The page contains **no verbatim quotes**,
so there is no quote-fabrication or misquote exposure on this page.

Two soft cross-page observations (defensible editorial simplifications, **not** flagged
as errors):

- **ANTLR placed under "Deterministic (LL / LR / PEG)" in the Split-1 table (L77), where
  the column's Time row reads O(n)** — while the catalog (L51) correctly gives ANTLR's
  ALL(_) the worst-case bound **O(n⁴)**. These do not contradict: ALL(_) is
  _deterministic in output_ (it commits to a single parse — "ambiguity is almost always
  an error", `theory-top-down` #36) and "consistently performs linearly … in practice"
  (`parr-2014` ln 54, re-checked). The split is about single-parse-commitment vs
  forest-output, not about worst-case asymptotics, so ANTLR belongs in the deterministic
  column. Worth a one-clause caveat only if the survey wants the O(n⁴)-worst-case nuance
  visible in the split table.

- **GLR worst-case given as "O(n³)" (L52, L75)** is the RNGLR/BRNGLR-improved cubic bound;
  Tomita's raw Algorithm 1 is O(n^{p+1}) in the longest RHS (`theory-bottom-up` #3/#48,
  web/secondary). The catalog uses the standard textbook cubic figure, consistent with the
  bottom-up deep-dive's own headline. No change needed.

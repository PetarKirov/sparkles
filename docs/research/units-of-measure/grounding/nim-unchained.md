  unit table (and note the var is `UnitTab`, type `UnitTable`). Not yet applied
  (coordinator to fix the page).

  row writes "`QuantityPower.power: int` (`ct_unit_types.nim` L31, `quantities.nim`
  L15–18)." `ct_unit_types.nim:31` is `UnitInstance.power*: int`, not
  `QuantityPower.power`; `QuantityPower.power` is `quantities.nim:17`. Both are
  `int` (the ℤ claim holds), but the ct_unit_types locator is mislabelled.
  **Fix:** attribute L31 to `UnitInstance.power` (or drop it), keep

## Discrepancies

- **#D1 ⚠ (minor) — `index.md:214`.** Milestone 1994: "ML Kit extension — the
  first implementation **of dimension inference**". The grounding page
  (`theory/kennedy-types.md:648`) says "**The first implementation**, an extension
  of the ML Kit Standard ML compiler" — i.e. the first implementation of
  _Kennedy's dimension-type system_; the page nowhere claims it was the first
  implementation of dimension inference generally (Wand & O'Keefe 1991 precede as
  theory, and the page is silent on any W&O implementation). The landing page's
  broader claim was not backed by its cited source. **Fix:** "ML Kit extension —
  the first implementation of Kennedy's dimension types". Fixed: yes (this pass).
